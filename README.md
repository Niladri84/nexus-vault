# Cyber Vault - Password Manager

A dark sci-fi themed password manager built with React and Tailwind CSS. Store, manage, and organize your passwords securely in your browser with a sleek cyberpunk aesthetic.

## Features

- **Add Passwords** - Store website credentials with notes
- **Edit & Delete** - Manage your password entries
- **Generate Passwords** - Create strong random passwords automatically
- **Show/Hide** - Toggle password visibility with eye icon
- **Copy to Clipboard** - One-click copying for usernames and passwords
- **Local Storage** - All data saved locally in your browser (no external servers)
- **Validation** - Form validation with error messages
- **Dark Sci-Fi Theme** - Cyberpunk aesthetic with cyan/purple gradients and animated backgrounds
- **Responsive Design** - Works on desktop and mobile

## Theme

Modern dark sci-fi aesthetic featuring:
- Gradient backgrounds (slate-950 to blue-950)
- Cyan and purple accent colors
- Glowing effects and animated gradients
- Grid background overlay
- Neon borders and subtle animations

## Preview

Runs on port 8080 for local development. Frontend and backend are configured for k8s HTTPRoute access.
